# SimAnt

A Godot 4 game. Click anywhere to spawn a red dot that moves to the right.

## How to Run

### 1. Install Godot 4

Download Godot 4.3+ from https://godotengine.org/download or install via Homebrew:

```bash
brew install --cask godot
```

### 2. Run from the Editor

Open Godot, click **Import**, select this project folder, then press **F5** (or the Play button).

### 3. Run from the Command Line

```bash
godot --path .
```

Or if Godot is installed as a `.app` on macOS:

```bash
/Applications/Godot.app/Contents/MacOS/Godot --path .
```
